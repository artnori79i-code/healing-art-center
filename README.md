# Healing Art Center - Interactive Web App

This is a high-motion, interactive React application for "Healing Art Center" (힐링예술터).

## Folder Structure
- `/index.html`: Entry page.
- `/css/`: CSS files.
- `/js/`: JavaScript/TypeScript logic including components.
- `/metadata.json`: Project metadata.

## Execution
1. Open this root directory in Antigravity or any IDE.
2. Run `npm install`.
3. Start the application with `npm start`.
4. Access via `http://localhost:3000`.

## Key Technologies
- **React 19**: Modern component-based UI.
- **Tailwind CSS**: Utility-first styling.
- **Framer Motion**: Fluid animations and transitions.
- **Gemini AI**: Intelligent consultant integration.
